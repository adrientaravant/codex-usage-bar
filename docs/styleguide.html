Coco design system · v1

Hard corners,
live data

One drop-in stylesheet — coco-design.css — distilled from the terminal-native landing page. Link it, then build with the tokens and classes below.

  • — Warm-violet near-black canvas, never neutral, never orange.
  • — Hard 90° corners everywhere. --radius: 0 and it stays 0.
  • — 1px hairlines divide; fills are nearly invisible.
  • — Mono-first — headlines, nav, eyebrows, and all data in Geist Mono.
  • — One violet brand accent + two data inks: teal = Codex, amber = Claude Code.
