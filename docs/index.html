Local first

Reads Codex and Claude Code JSONL logs locally. Claude limits use your existing Claude Code OAuth credentials from Keychain.

Minimal by design

No webview, no analytics, no server, no account system. The app caches a small snapshot and refreshes in the background.

Install from source

Clone the repo and run ./script/install.sh --login if you prefer to build the exact binary on your Mac.
